# xnat_pipelines Examples

This directory is packaged with the wheel so example schemas and the monitoring notebook are installed alongside the library.

- `debug_command/` – example schema for `xnat/debug-command`
- `dcm2niix/` – example schema for `xnat/dcm2niix`
- `xnat_pipelines_monitoring_jupyterhub.ipynb` – JupyterHub-friendly live monitoring notebook
